
# Docker セットアップ
**Dockerイメージの作成からコンテナの作成、起動**  
Dockerコンテナが作成済の場合、再構築のためbuildコマンドが必要  
`docker compose up`

**Dockerイメージを作成**  
`docker compose build`

**Dockerコンテナを起動**  
`docker compose start	`
